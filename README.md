# sound-activated-voice-chip-library
Sound Activated Voice Chip for Library - detects noise and gives voice alert to maintain silence

## Overview
This project is an embedded system designed to maintain silence in libraries. 
The system detects sound using a microphone sensor. When the sound exceeds a threshold level, 
a microcontroller activates a voice chip that plays a recorded message such as:

"Silence Please".

This helps maintain a quiet environment automatically.

## Features
- Automatic noise detection
- Voice warning system
- Embedded microcontroller based design
- Low cost and simple implementation

## Hardware Components
- Condenser Microphone
- Microcontroller (89C2051)
- Audio Amplifier
- Op-Amp Comparator
- Voice Record and Playback Chip
- Speaker
- Power Supply (+5V)

## Applications
- Libraries
- Study halls
- Silent zones
- Examination halls
